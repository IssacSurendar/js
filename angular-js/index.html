1. Intro

  • JS Framework that helps to build web application.
  • Suitable for Single Page Application
  • Extends html attribute with directives and bind data with expression

    • 2. Expression

  • Written inside double brace. {{expression}}
  • Written also inside directive. ng-bind="expression"
    • String
    <div ng-app="" ng-init="myCol='lightblue'"> <input style="color:{{myCol}}" ng-model="myCol"> </div>
    Number
    expression: {{qty * cost}}   Bind:
    <div ng-app="" ng-init="quantity=1;cost=5"> <p>{{ quantity * cost }}</p> <p><span ng-bind="quantity * cost"></span></p> </div>
    Object
    Expression: {{person.fn}}   Bind:
    <div ng-app="" ng-init="person={firstName:'John',lastName:'Doe'}"> <p>{{ person.lastName }}</p> <p><span ng-bind="person.lastName"></span></p> </div>
    Array
    Expression: {{points[0]}}  Bind:
    <div ng-app="" ng-init="points=[1,15,19,2,40]"> <p>{{ points[2] }}</p> <p><span ng-bind="points[2]"></span></p> </div>

    3. Modules

  • Module defines an application.
  • Module is a container for the different parts of an application.
    • Controller
    {{firstName+" "+lastName}} Bind :
    <div ng-app="myApp" ng-controller="myCtrl"> {{ firstName + " " + lastName }} </div> <script> var app = angular.module("myApp", []); app.controller("myCtrl", function($scope) { $scope.firstName = "John"; $scope.lastName = "Doe"; }); </script>
    Directives
    <div ng-app="myApp" w3-test-directive></div> <script> var app = angular.module("myApp", []); app.directive("w3TestDirective", function() { return { template : "I was made in a directive constructor!" }; }); </script>
    Repeat Array
    • {{ i }}
    <div ng-app="myApp" ng-init="names=['surendar', 'issac', 'karthick']"> <li ng-repeat="x in names">{{}}</li> <li ng-repeat="x in names"><span ng-bind="x"></span></li> </div>
  • {{x.name+x.age}}
    • <div ng-app="myApp" ng-init="names=[ {name:'surendar', age: 12}, {name:'issac', age: 16} ]"> <ul> <li ng-repeat="x in names">{{x.name+" "+x.age}}</li> <li ng-bind="x in names"><span ng-bind="x.name"></span> <span ng-bind="x.age"></span></li> </ul> </div>

    Directives

  • ng-app Intialize as an application.
  • ng-init Intialize data.
  • ng-bind Bind the input value to data.
  • ng-repeat Repeat the html element. (foreach)
